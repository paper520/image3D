前要说明

基本的绘图步骤我们在 快速开始 中给出了具体的例子，这里不再赘述。这里主要是想对教程的主体和将说明的内容，包括代码结构进行说明。

本教程主要说明的内容包括以下部分：

  • 第一部分：image3D.js的使用（主要是核心接口和辅助方法）；
  • 第二部分：WebGL重要的原生知识或接口；
  • 第三部分：三维绘图的技巧和实践校验。

以上三部分并不会独立的提出来说明，而是穿插在整个文档中来说明，其目的只有一个：借助image3D绘制有趣的图形！

代码结构

我们给出一份简洁的代码结构，后续所有的用例都可以基于此开始开发： 

<!DOCTYPE html>
<html lang="zh-cn">
<head>
    <script src="build/image3D.js"></script>
    <!-- 顶点着色器 -->
    <script type='x-shader/x-vertex' id='vs'>
        void main(){
           // todo
        }
    </script>
    <!-- 片段着色器 -->
    <script type='x-shader/x-fragment' id='fs'>
        void main(){
            // todo
        }
    </script>
</head>
<body>
    <canvas width=500 height=500>非常抱歉，您的浏览器不支持canvas!</canvas>
</body>
<script>
    var render3D = $$(document.getElementsByTagName('canvas')[0]).render3D();

    // 启用着色器
    render3D.shader(
        document.getElementById('vs').innerHTML,
        document.getElementById('fs').innerHTML
    );

    // 获取画笔
    var gl = render3D.painter();

   // todo

</script>
</html>

当然，以上结构仅供参考，具体的结构可根据具体情况而定（比如着色器，可以提前预定义好各种着色器字符串，根据情况选择加载）。
