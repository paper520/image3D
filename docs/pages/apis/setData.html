数据传递

1.设置值或向量 

image3d.setAttributeFloat(location, v0, v1, v2, v3);

设置Attribute类型的Float类型值或向量（v1~v3均可选，会根据最终输入的参数长度自动判断设置的值类型或向量长度，下同）。 

image3d.setAttributeInt(location, v0, v1, v2, v3);

设置Attribute类型的Int类型值或向量。 

image3d.setUniformFloat(location, v0, v1, v2, v3);

设置Uniform类型的Float类型值或向量。 

image3d.setUniformInt(location, v0, v1, v2, v3);

设置Uniform类型的Int类型值或向量。

2.设置矩阵 

image3d.setUniformMatrix(location, value);

设置Uniform类型的矩阵，value应该是一个数组，会自动根据此数组的长度来判断是2x2、3x3还是4x4尺寸。
