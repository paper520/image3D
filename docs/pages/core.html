var core = image3D.core(document.getElementsByTagName('canvas')[0]);

通过调用image3D上的core方法，传递一个canvas结点，返回的是一个对象，记作core，通过它你可以直接使用image3D提供的最基础的原始方法。

着色器

<!-- 顶点着色器 -->
<script type='x-shader/x-vertex' id='vs'>
    void main(){}
</script>
<!-- 片段着色器 -->
<script type='x-shader/x-fragment' id='fs'>
    void main(){}
</script>

在JavaScript层面上着色器就是二段字符串，调用下面的便可以让着色器生效： 

core.shader(
    document.getElementById('vs').innerHTML,
    document.getElementById('fs').innerHTML
);

设置数据

我们知道uniform是只读的，可以是除了数组和结构体外的任意类型，主要用于二个着色器间数据传递，因此，设置数据其实涉及的是attribue和varying类型的变量（只能是float、vec2、vec3、vec4和mat2、mat3、mat4其中的一种类型）。

1.attribue

core.setAttribute[1,2,3,4][f,i](location, v0, v1, v2, v3);

2.varying

core.setUniform[1,2,3,4](location, v0, v1, v2, v3);

此外，如果设置的数据是矩阵： 

core.setUniformMatrix[2,3,4][f,i](location, value);

缓冲区

缓冲区是一种更快速的给GPU大量数据的特殊通道，为了使用缓冲区，首先需要创建一个缓冲区： 

var buffer = core.buffer(isElement)

在创建的时候需要传递一个boolean值类型的参数isElement，默认false表示缓冲区中保存了包含顶点的数据，如果传递true表示缓冲区保存了包含顶点的索引值。

缓冲区创建好了以后，你就可以直接往缓冲区里面写入数据了： 

buffer.write(data);

如果你创建的缓冲区是包含顶点的索引值的话，可能到这一步就结束了，否则，你可能还需要把缓冲区中的数据分配给具体的使用者： 

buffer.use(location, size, stride, offset);
  • 1.location：字符串类型，对应顶点着色器中定义的attribute类型的变量名；
  • 2.size：整数，表示一个完整的数据的个数；
  • 3.stride：整数，表示写入缓冲区数据一组的个数；
  • 4.offset：整数，表示读取起点。

纹理

纹理是片元着色器在逐片元过程中使用的，类似贴图，使用的第一步也是创建一个纹理： 

var texture = core.texture(unit, type)

其中unit表示使用哪个纹理单元（开启的编号）；type表示纹理的类型，因为纹理有很多种，下面列出目前支持的类型：

  • 1."2d"：二维纹理(默认值)；
  • 2."cube"：立方体纹理。

纹理创建好了以后，你需要把材料（比如图片等）和纹理绑定（假如我们这里已经准备好了一张图片img）： 

texture.use(img);

到这一步，纹理准备工作已经完成了，后面的就是在片元着色器中去使用这个纹理了。

绘图方法

var painter=core.painter();

绘制方法可以划分为二类：顶点绘制方法和顶点索引绘制方法。

1.顶点绘制

点：

painter.points(first, count);

直线：

painter.lines(first, count);

连续直线：

painter.stripLines(first, count);

闭合直线：

painter.loopLines(first, count);

三角形：

painter.triangles(first, count);

共边三角形：

painter.stripTriangles(first, count);

旋转围绕三角形：

painter.fanTriangles(first, count);
